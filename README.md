# STM32 遥控小车项目

## 简介
本项目实现了一个基于 STM32F103C8 微控制器的遥控小车，采用蓝牙模块进行无线控制，舵机模块实现转向控制，直流电机模块实现驱动。

## 硬件配置

### 主控芯片
- **型号**: STM32F103C8

### 蓝牙模块
- **型号**: HC-05
- **计时器**: USART3
- **引脚连接**:
  - STM32 B10 <--> TX (HC-05)
  - STM32 B11 <--> RX (HC-05)

### 舵机转向模块
- **计数器**: TIM2
- **引脚连接**:
  - STM32 A0 <--> PWM (舵机)

### 直流电机模块
- **型号**: TB6612FNG
- **计数器**: TIM1
- **引脚连接**:
  - STM32 A12 <--> PWMA
  - STM32 B8  <--> AIN1
  - STM32 B9  <--> AIN2
  - STM32 A10 <--> PWMB
  - STM32 B5  <--> BIN2
  - STM32 B6  <--> BIN1
