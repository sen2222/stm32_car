      stm32   B10  <--------->  TX    HC_05

  引脚连接:
      stm32   A0  <--------->  PWM    舵机
========================================================

      stm32  A12 <----------> PWMA
             B3  <----------> AIN1
             B4  <----------> AIN2
             A10 <----------> PWMB
             B5  <----------> BIN2
             B6  <----------> BIN1
